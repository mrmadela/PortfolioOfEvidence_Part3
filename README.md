## PROJECT DESCRIPTION

This application is a Cybersecurity Awareness Chatbot developed in C# using WPF.

The chatbot provides cybersecurity information and includes the following features:

• User login and greeting system
• Cybersecurity awareness chatbot
• Natural Language Processing (NLP) for intent recognition
• Task Manager for creating and managing cybersecurity tasks
• Reminder management
• Cybersecurity Quiz
• Activity Log tracking user actions
• MySQL database integration for task storage



## SOFTWARE REQUIREMENTS

• Visual Studio 2022
• .NET Framework
• MySQL Server 8.0 or later
• MySQL Workbench
• MySql.Data NuGet Package



## DATABASE SETUP

1. Install MySQL Server and MySQL Workbench.

2. Open MySQL Workbench and connect to your local server.

3. Open the file:

   cybertasksdb.sql

4. Execute the SQL script.

5. Verify that the database "cybertasksdb" has been created.

6. Open task_manager.cs and update the connection string if necessary:

   server=localhost;
   database=cybertasksdb;
   uid=root;
   pwd=YOUR_PASSWORD;

7. Build and run the project in Visual Studio.



## DEFAULT DATABASE TABLE

The application automatically creates the Tasks table if it does not already exist when the application starts.
